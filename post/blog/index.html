小白博客搭建指南（ hugo 博客开源框架）

1 windows安装

1.1 安装 git （设置好自己GitHub上的用户名及邮箱）

  1. 安装 git

    • 参考网址：https://www.cnblogs.com/ximiaomiao/p/7140456.html

  2. 打开 cmd 终端，输入 git 检查 git 是否安装成功

  3. 配置 git 环境变量

  4. 检查 git 是否安装成功，环境变量配置成功

  1. git 设置默认邮箱及用户名

    • 打开 git 终端，Git Bash
    • git config –global user.name 你的目标用户名；
    • git config –global user.email 你的目标邮箱名;

1.2 安装 hugo 开源框架

  1. 下载并安装 hugo 开源框架
  2. 访问地址：https://github.com/gohugoio/hugo/releases
  3. 看自己情况下载 32 位或 64 安装包

  1. 将压缩包解压到软件安装位置，并配置换将变量

  1. 配置 hugo 框架的环境变量

  1. 打开 cmd 终端，输入 hugo version 检查 hugo 是否安装配置成功，出现版本号代表成功

  1. 创建第一篇博客

    1. hugo new site myblog
    2. 生成的文件位置在 c 盘/用户/myblog(此为其根目录)

  2. 选择 hugo 主题进行下载

    1. https://themes.gohugo.io/

    2. 选择中意的主题，找到其 git 地址

    3. 进入终端，进入 hugo 根目录，进行主题下载

      1. 根目录 myblog 里面一层

      2. git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

      3. 如果出现一下情况

        • Cloning into ‘themes/m10c’…

      fatal: unable to access ‘https://github.com/vaga/hugo-theme-m10c.git/‘: error:1407742E:SSL routines:SSL23_GET_SERVER_HELLO:tlsv1 alert protocol version

      将上面命令 https 替换为 git

    4. 使用主题并发布在本地，执行命令：hugo server -t m10c –buildDrafts

    5. 访问其默认生成的本地网址进行访问：http://localhost:1313

    6. 根目录下新建一篇文章

      1. 进入 myblog 目录

      2. 执行命令：hugo new post/blog.md

      3. 会在根目录 content 文件夹下 生成其文件 ：C:\Users\sunshijian\myblog\content\post
